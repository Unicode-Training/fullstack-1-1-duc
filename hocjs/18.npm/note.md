# npm = node package manager

1. Khởi tạo file package.json

npm init -y

2. Cài thư viện

npm i

npm i tenthuvien

npm i tenthuvien@phienban

3. Xóa bỏ thư viện

npm un tenthuvien

4. Cập nhật thư viện

npm update
npm update tenthuvien

5. Cài thư viện dành cho dev

npm i tenthuvien --save-dev

6. Chỉ cài đặt các thư viện production

npm i --production

Trình duyệt muốn dùng các thư viện trong node_modules --> Dùng build tool (Bộ công cụ dành cho dev giúp chuyển node_modules có thể dùng ở trình duyệt, dev server, hot reload, build product --> deploy server)
